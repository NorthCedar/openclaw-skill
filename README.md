# openclaw-skill

OpenClaw 自定义技能仓库 —— 黄金藤椒大鸡腿的技能集合与设计文档。

## 结构

```
skills/          # 技能目录（每个技能一个子文件夹）
docs/            # 设计文档、架构笔记
```

## 约定

- 每个技能遵循 [AgentSkills 规范](https://docs.openclaw.ai/skills)
- 技能目录结构：`skills/<skill-name>/SKILL.md` + 可选 `scripts/`、`references/`
- 设计文档放 `docs/`，按主题命名
